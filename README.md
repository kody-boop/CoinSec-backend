# CoinSec - 多端加密记账系统

CoinSec 是一款专注于隐私保护的跨平台记账系统，支持 iOS、Android、PC 多端同步，采用端到端加密技术保障用户财务数据安全，同时提供灵活的账本管理、分类统计和数据同步功能。

## 核心特性

- **隐私优先**：核心数据端到端加密，云端仅存储密文，本地数据加密存储
- **多端同步**：基于版本控制的增量同步机制，确保多设备数据一致性
- **高度自定义**：支持自定义分类、记账模板和统计维度
- **数据安全**：采用 HTTPS 传输、AES-256 加密、SQLCipher 本地加密等多层安全防护
- **灵活管理**：多账本隔离、分类层级管理、预算设置等功能

### 后端技术栈（云同步服务）

- **框架**：Spring Boot
- **数据库**：PostgreSQL（云端存储）
- **辅助工具**：
    - 日志：SLF4J + Logback
    - 接口文档：SpringDoc OpenAPI
- **加密支持**：BouncyCastle（AES-256 算法支持）

### 安全与加密

- **传输安全**：HTTPS/TLS 1.3
- **数据加密**：AES-256 端到端加密（前端加密，后端仅存密文）
- **密钥管理**：PBKDF2 算法生成密钥（基于用户密码 + 设备标识）
- **本地安全存储**：

    - 移动端：iOS Keychain / Android Keystore
    - PC 端：加密 LocalStorage

## 核心功能模块

1. **用户认证**：基于邮箱的注册/登录，JWT 令牌身份验证
2. **账本管理**：多账本创建、切换、权限控制，支持预算设置
3. **分类管理**：多级收支分类，支持自定义分类属性
4. **记账记录**：加密存储收支明细，支持自定义字段和附件
5. **数据同步**：增量同步机制，支持断点续传和冲突处理
6. **数据统计**：基于时间、分类、支付方式等多维度统计
